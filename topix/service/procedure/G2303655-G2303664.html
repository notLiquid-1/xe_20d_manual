Removal
WARNING:

Be prepared to collect escaping oil.

CAUTION:

Make sure that all components and tools are clean and free from foreign material.

NOTES:

  • This procedure contains some variation in the illustration depending on the vehicle specification, but essential information is always correct.

  • This procedure contains illustrations showing certain components removed to provide extra clarity.


  1. Raise and support the vehicle on a suitable 2 post lift.
    Refer to: Lifting (100-02 Jacking and Lifting, Description and Operation).
    General Equipment: 2 post lift
  2. Drain the rear differential oil.
    Refer to: Differential Draining and Filling vehicles with heavy duty differential (205-02 Rear Drive Axle/Differential, General Procedures).
  3. Remove the rear driveshaft.
    Refer to: Driveshaft - V6 S/C 3.0L Petrol (205-01 Driveshaft, Removal and Installation).
  4. NOTE:

    Measure for installation.

    Measure the depth of the pinion nut on the pinion shaft.
  5. Mark the drive pinion shaft in relation to the drive flange.
  6. Using the special tool, remove the drive pinion flange.
    Special Tool(s): 205-053, 205-053-03
  7. Install the differential pinion flange remover.
    Special Tool(s): JLR-205-1014, 204-269A
  8. NOTE:

    Be prepared to collect escaping fluids.

    Carefully remove and discard the drive pinion oil seal.
Installation
  1. Clean the drive pinion seal mating faces.
  2. CAUTION:

    Make sure the seal is installed to the correct end of the special tool.

    • Using the special tools, install the new drive pinion seal.
    • Make sure that the seal is fully installed and the seal installer tool evenly touches the differential case.
      Special Tool(s): JLR-307-520-01, JLR-205-998
  3. Install the drive pinion flange. Make sure the marks are correctly aligned.
  4. CAUTIONS:

    • Make sure the drive pinion flange has no end float and is free to rotate.

    • Make sure that the drive pinion flange scribed mark is aligned and is never tightened short of the scribed mark on the drive pinion shaft and is no more than a maximum of 5 degrees past the scribed mark on the drive pinion shaft.

    NOTES:

    • Using the special tool, counter hold the drive pinion flange.

    • Measure the depth of the pinion nut on the pinion flange.

    Using the special tool, install the drive pinion flange.
    Special Tool(s): 205-053, 205-053-03
  5. Install the rear driveshaft.
    Refer to: Driveshaft - V6 S/C 3.0L Petrol (205-01 Driveshaft, Removal and Installation).
  6. Fill the rear differential.
    Refer to: Differential Draining and Filling vehicles with heavy duty differential (205-02 Rear Drive Axle/Differential, General Procedures).
