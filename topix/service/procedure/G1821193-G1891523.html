Removal
NOTES:

  • Engine shown removed for clarity.

  • Removal steps in this procedure may contain installation details.


  1. Disconnect the battery ground cable.
    Refer to: Startup Battery Disconnect and Connect (414-01 Battery, Mounting and Cables, General Procedures).
  2. WARNING:

    Make sure to support the vehicle with axle stands.

    Raise and support the vehicle.
  3. Remove the transmission.
    Refer to: Transmission - GTDi 2.0L Petrol (307-01 Automatic Transmission/Transaxle - Vehicles With: 8HP45 8-Speed Automatic Transmission RWD, Removal).
  4. Install the special tool.
  5. CAUTION:

    Only tighten the bolt finger-tight at this stage.

  6. CAUTION:

    Make sure that the special tool is correctly located.

  7. Torque: 90 Nm
  8. NOTE:

    Loosen the bolts, but do not fully remove.

    • Special Tool(s): JLR-303-1594
  9. Loosen the bolt (1) remove the bolts (2)
  10. CAUTION:

    Discard the bolts.

    Remove the components.
Installation
  1. CAUTIONS:

    • Only tighten the bolts finger-tight at this stage.

    • Make sure that new bolts are installed.

    Renew Part: Flexplate bolts Quantity: 1.
  2. Install the special tool.
  3. CAUTION:

    Only tighten the bolt finger-tight at this stage.

  4. CAUTION:

    Make sure that the special tool is correctly located.

  5. CAUTION:

    Make sure that the special tool is correctly located.

    Torque: 90 Nm
  6. CAUTION:

    Make sure that new bolts are installed.

    NOTE:

    Tighten the retaining bolts in the sequence shown.

    Torque:
    Stage 1: 50 Nm
    Stage 2: 70 Nm
    Stage 3: 112 Nm
  7. Loosen the bolt (1) remove the bolts (2)
  8. Install the transmission.
    Refer to: Transmission - GTDi 2.0L Petrol (307-01 Automatic Transmission/Transaxle - Vehicles With: 8HP45 8-Speed Automatic Transmission RWD, Installation).
  9. Connect the battery ground cable.
    Refer to: Startup Battery Disconnect and Connect (414-01 Battery, Mounting and Cables, General Procedures).
