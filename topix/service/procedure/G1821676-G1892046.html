Removal
NOTE:
  • This procedure contains some variation in the illustrations depending on the vehicle specification, but the essential information is always correct.
  • This procedure contains illustrations showing certain components removed to provide extra clarity.

  1. Raise and support the vehicle using a suitable 2 post lift.
    Refer to: Lifting (100-02 Jacking and Lifting, Description and Operation).
  2. Remove the engine cover.
    Refer to: Engine Cover - INGENIUM I4 2.0L Diesel (501-05 Interior Trim and Ornamentation, Removal and Installation).
  3. Remove the bolt and release the clips.
  4. Release the clip.
  5. Remove the air deflector.
    Refer to: Engine Undershield (501-02 Front End Body Panels, Removal and Installation).
  6. Remove the bolts and release the lever.
  7. Release the clips.
  8. Remove the emergency park position release lever.
  9. Release clip from the bracket.
Installation
  1. Secure the emergency park postilion release lever to the bracket.
  2. Install the emergency park postilion release lever
  3. NOTE:

    If necessary, install new clips.

    Secure the clips.
  4. install the bolts and secure the lever.
    Torque: 11 Nm
  5. Install the air deflector.
    Refer to: Engine Undershield (501-02 Front End Body Panels, Removal and Installation).
  6. NOTE:

    If necessary, install new clips.

    Secure the clip.
  7. NOTE:

    If necessary, install new clips.

    Install the bolt and Secure the clips.
    Torque: 12 Nm
  8. install the engine cover.
    Refer to: Engine Cover - INGENIUM I4 2.0L Diesel (501-05 Interior Trim and Ornamentation, Removal and Installation).
