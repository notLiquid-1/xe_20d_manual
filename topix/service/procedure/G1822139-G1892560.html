Removal
NOTES:

  • This procedure contains illustrations showing certain components removed to provide extra clarity.

  • This procedure contains some variation in the illustrations depending on the vehicle specification, but the essential information is always correct.


  1. Remove the windshield.
    Refer to: Windshield (501-11 Glass, Frames and Mechanisms, Removal and Installation).
  2. NOTE:

    Repeat the procedure for the other side.

    Remove the B-pillar upper trim.
    Refer to: B-Pillar Upper Trim Panel (501-05 Interior Trim and Ornamentation, Removal and Installation).
  3. NOTE:

    Repeat the procedure for the other side.

    Remove the C-pillar trim panel.
    Refer to: C-Pillar Trim Panel (501-05 Interior Trim and Ornamentation, Removal and Installation).
  4. Lower both front row seat backrests.
  5. NOTE:

    Repeat the step for the other side.

    • Disconnect the electrical connector.
    • Release the wiring harness.
  6. NOTE:

    Repeat the step for the other side.

    Remove the sun visor.
    Refer to: Sun Visor (501-05 Interior Trim and Ornamentation, Removal and Installation).
  7. NOTE:

    Do this step for all 4 components.

    Open the access covers.
  8. NOTE:

    Do this step for all 4 components.

    • Remove the 2 bolts.
    • Release the rear overhead console.
    • Disconnect the electrical connector.
    • Remove the rear overhead console.
    • Disconnect the 3 electrical connectors.
    • Remove the front overhead console.
  12. Disconnect the electrical connector at the left C-pillar.
    • Release the 3 front clips.
    • Release the 2 magnets.
    • Release the 3 center clips.
  14. NOTE:

    This step requires the aid of another technician.

    • Release the 4 center brackets.
    • With assistance, carefully lower the headliner.
  15. CAUTION:

    Care must be taken not to damage the headliner on removal.

    NOTE:

    This step requires the aid of another technician.

    • Tilt the steering wheel.
    • With assistance, carefully remove the headliner.
  16. Remove the clip from the roof panel.
  17. NOTES:

    • Only do the following step(s) if a new component is to be installed.

    • Do this step for all 4 components.

    Release the 4 clips to remove the 4 passenger assist handles.
    • Disconnect the electrical connector.
    • Release the 2 clips to remove the Tire Pressure Monitoring System (TPMS) receiver.
    • Disconnect the electrical connector.
    • Release the 2 clips to remove the radio frequency receiver.
  20. NOTE:

    If equipped.

    • Disconnect the electrical connector.
    • Remove the 2 screws.
    • Remove the headphone transmitter.
Installation
  1. NOTE:

    If equipped.

    • Install the headphone transmitter.
    • Install and tighten the 2 screws.
      Torque: 2 Nm
    • Connect the electrical connector.
    • Install the radio frequency receiver.
    • Connect the electrical connector.
    • Install the Tire Pressure Monitoring System (TPMS) receiver.
    • Connect the electrical connector.
    • Install the 4 passenger assist handles.
  5. Install the clip to the headliner.
  6. NOTE:

    This step requires the aid of another technician.

    • Install the headliner into the vehicle.
  7. NOTE:

    This step requires the aid of another technician.

    Install the headliner rear clip to the roof panel.
    • With assistance, install the headliner 3 center clips.
    • Install the 2 magnets.
    • Install the 4 brackets to the opening panel top frame.
  9. Install the headliner 3 front clips.
  10. Connect the electrical connector at the left C-pillar.
    • Connect the 3 electrical connectors to the front overhead console.
    • Install the front overhead console.
    • Connect the electrical connector to the rear overhead console.
    • Install the rear overhead console.
    • Install and tighten the 8 passenger assist handle bolts.
      Torque: 5.9 Nm
    • Close the access covers.
  14. NOTE:

    Repeat the step for the other side.

    Install the sun visor.
    Refer to: Sun Visor (501-05 Interior Trim and Ornamentation, Removal and Installation).
  15. NOTE:

    Repeat the step for the other side.

    • Connect the electrical connector at the left A-pillar.
    • Secure the 2 electrical harness clips.
  16. NOTE:

    Repeat the procedure for the other side.

    Install the C-pillar trim panel.
    Refer to: C-Pillar Trim Panel (501-05 Interior Trim and Ornamentation, Removal and Installation).
  17. NOTE:

    Repeat the procedure for the other side.

    Install the B-pillar upper trim.
    Refer to: B-Pillar Upper Trim Panel (501-05 Interior Trim and Ornamentation, Removal and Installation).
  18. Install the windshield.
    Refer to: Windshield (501-11 Glass, Frames and Mechanisms, Removal and Installation).
  19. Reposition the front row seat backrest to the original position.
