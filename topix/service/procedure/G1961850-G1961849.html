Front End Sheet Metal Repairs

Headlamp Mounting Support Bracket (G1961850)

REMOVAL AND INSTALLATION

Loading...

Removal
NOTE:

In this procedure the headlamp mounting support bracket is installed in conjunction with:

  • Front bumper
  • Front armature
  • Headlamp mounting bracket
  1. The headlamp mounting support bracket is serviced as a indicated.
  2. Before commencing this procedure make sure that you are aware of all Health and Safety requirements. For additional information, refer to: Health and Safety Precautions (100-00 General Information, Description and Operation).
  3. For further information on the methods, tools and fixings used in this procedure refer to the body repairs - general information section.
    4. NOTES:

    • Where breakstem fasteners (BSF) removal is instructed remove fastener shaft with a 4mm punch and remove the head with a 6.5mm drill bit. Clean all the drilled holes to make sure they are free from debris.

    • Where self piercing rivets (SPR's) removal is instructed use either the approved SPR tool or 5.3mm drill bit.

    • Any joints sealed as part of vehicle manufacture must be sealed as part of the repair.

  4. For additional information, refer to: Front Bumper (501-19 Bumpers, Removal and Installation).
  5. For additional information, refer to: Front Armature (501-19 Bumpers, Removal and Installation).
  6. For additional information, refer to: Headlamp Mounting Bracket (501-02 Front End Body Panels, Removal and Installation).
  7. Remove the SPR's as indicated.
  8. Carefully separate the joints and remove the headlamp mounting support bracket.
Installation
CAUTION:

Where structural adhesive is to be applied between surfaces it is essential compression is applied to the joints during the curing period.

NOTES:

  • BSF installation involves using the Jaguar approved BSF installation tool and requires a hole diameter of 6.5mm.

  • SPR installation requires using the Jaguar approved SPR tool.

  1. Remove rivet remnants.
  2. Dress the flanges where necessary.
  3. Using a fine bristle disc clean and prepare the panel surfaces.
  4. Apply the coupling agent where the Jaguar recommended bonding material is to be applied and allow to dry.
  5. NOTE:

    Make sure a continuous bead of adhesive surrounds fixing holes.

    Apply a 5mm zig zag bead of the Jaguar recommended bonding material as indicated.
  6. Offer up the new headlamp mounting support bracket and clamp into position. Check alignment if correct proceed to next step, if not rectify and recheck before proceeding.
  7. NOTE:

    Make sure that the SPR's are installed from inside the front side member.

    Install the SPR's as indicated.
  8. Remove any excess adhesive.
  9. Make sure that any open or exposed panel joints are correctly sealed.
  10. Make sure that all installed SPR's are sealed using a Jaguar recommended sealer.
  11. Make sure corrosion protection is applied to all areas affected by repair.
  12. The installation of associated panels and components is the reversal of removal procedure.
