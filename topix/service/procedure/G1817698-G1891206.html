Removal
CAUTION:

To prevent any stress being applied to the brake hose, make sure the suspension is supported before disconnecting the rear shock absorber from the hub.

NOTE:
  • This procedure contains some variation in the illustrations depending on the vehicle specification, but the essential information is always correct.
  • This procedure contains illustrations showing certain components removed to provide extra clarity.

  1. Raise the vehicle on a suitable 2 post lift.
    Refer to: Lifting (100-02 Jacking and Lifting, Description and Operation).
  2. Remove the wheel and tire.
    Refer to: Wheel and Tire (204-04 Wheels and Tires, Removal and Installation).
  3. Using a suitable stand, support the lower arm.
  4. Remove the 2 retaining clips and position the arch liner as illustrated.
  5. NOTE:

    Vehicles with adaptive damping.

    Disconnect the electrical connector from the adaptive damper.
  6. Remove and discard the upper shock absorber nuts.
  7. Remove and discard the lower shock absorber nut and bolt.
  8. NOTES:

    • Do not disassemble further if the component is removed for access only.

    • This step is not necessary if removed for access only.

    Remove and discard the nut.
Installation
  1. NOTE:

    This step is only required if previously removed.

    • Inspect the components and renew if damaged or worn.
    • Install the components as illustrated.
    • Tighten the nut to the correct torque.
      Renew Part: Shock absorber nut Quantity: 1.
      Torque:
      Active dampers 27 Nm
      Passive dampers 40 Nm
  2. CAUTION:

    Nuts and bolts must be tightened with vehicle at normal ride height.

    • Install the shock absorber.
    • Tighten the lower nut and bolt to the correct torque.
      Renew Part: Shock absorber to wheel knuckle mounting nut and bolt Quantity: 2.
      Torque:
      Stage 1: 70 Nm
      Stage 2: 120°
  3. Install the upper shock absorber nuts and tighten to the correct torque.
    Renew Part: Shock absorber upper mounting nuts Quantity: 2.
    Torque:
    Stage 1: 48 Nm
    Stage 2: 60°
  4. NOTE:

    Vehicles with adaptive damping.

    Connect the adaptive damper electrical connector.
  5. Reposition the wheel arch liner and install the 2 retaining clips.
  6. Remove the supporting stand.
  7. Install the wheel and tire.
    Refer to: Wheel and Tire (204-04 Wheels and Tires, Removal and Installation).
