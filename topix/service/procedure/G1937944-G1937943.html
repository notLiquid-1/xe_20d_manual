  1. The side panel reinforcement front assembly is serviced as indicated.
  2. Before commencing this procedure make sure that you are aware of all Health and Safety requirements. For additional information, refer to: Health and Safety Precautions (100-00 General Information, Description and Operation).
  3. For further information on the methods, tools and fixings used in this procedure refer to the body repairs - general information section.
    4. NOTES:

    • Where breakstem fasteners (BSF) removal is instructed remove fastener shaft with a 4mm punch and remove the head with a 6.5mm drill bit. Clean all the drilled holes to make sure they are free from debris.

    • Where self piercing rivets (SPR) removal is instructed use either the approved SPR tool or 5.3mm drill bit.

    • Any joints sealed as part of vehicle manufacture must be sealed as part of the repair.

Installation

All vehicles

CAUTION:

Where structural adhesive is to be applied between surfaces it is essential compression is applied to the joints during the curing period.

NOTES:

  • BSF installation involves using the Jaguar approved BSF installation tool and requires a hole diameter of 6.5mm.

  • SPR installation requires using the Jaguar approved SPR tool.

  1. Remove the rivet remnants.
  2. Dress flanges where necessary.
  3. Offer up the new side panel reinforcement front assembly and clamp into position. Check alignment if correct proceed to next step, if not rectify and recheck before proceeding.
  4. Drill holes where the BSFs are to be installed as indicated.
  5. Drill holes where the BSFs are to be installed as indicated.
  6. Drill holes where the BSFs are to be installed as indicated.
  7. Drill holes where the BSFs are to be installed as indicated.

Vehicles with roof opening panel

  1. CAUTION:

    When installing the roof panel in conjunction with the side panel reinforcement front assembly. The following step must be followed.

    NOTE:

    Left side and right side similar.

    Drill a hole where the BSF is to be installed as indicated.

Vehicles without roof opening panel

  1. CAUTION:

    Install the BSF before installation of the roof panel. Vehicles without roof opening panel only.

    NOTE:

    Left side and right side similar.

    Drill a hole where the BSF is to be installed as indicated.

All vehicles

  1. Remove the new side panel reinforcement front assembly.
  2. Deburr the drilled holes.
  3. Using a fine bristle disc clean and prepare the panel surfaces.
  4. Apply the coupling agent where the Jaguar recommended bonding material is to be applied and allow to dry.
  5. NOTE:

    Make sure a continuous bead of adhesive surrounds fixing holes.

    Apply a 5mm zig zag bead of the Jaguar recommended bonding material as indicated.
  6. Offer up the new side panel reinforcement front assembly and clamp into position. Check alignment if correct proceed to next step, if not rectify and recheck before proceeding.
  7. Install the BSFs as indicated.
  8. Install the BSFs as indicated.
  9. Install the BSFs as indicated.

Vehicles with roof opening panel

  1. NOTE:

    Make sure that the BSFs are installed from the inside.

    Install the BSFs and the SPRs as indicated.
  2. CAUTION:

    When installing the roof panel in conjunction with the side panel reinforcement front assembly. The following step must be followed.

    NOTE:

    Left side and right side similar.

    Install the BSF through the roof panel as indicated.For additional information, refer to: Roof Panel - Vehicles With: Roof Opening Panel (501-28 Roof Sheet Metal Repairs, Removal and Installation).
    3. NOTE:

    Check Breakstem Fastener and dress any excess material.

  3. CAUTION:

    When installing the roof panel in conjunction with the side panel reinforcement front assembly. The following step must be followed.

    NOTE:

    Left side and right side similar.

    Install additional roof opening panel frame spacers as indicated.For additional information, refer to: Roof Opening Panel Frame (501-17 Roof Opening Panel, Removal and Installation).

Vehicles without roof opening panel

  1. NOTE:

    Make sure that the BSFs are installed from the inside.

    Install the BSFs and the SPR's as indicated.
  2. CAUTION:

    Install the BSF before installation of the roof panel. Vehicles without roof opening panel only.

    NOTE:

    Left side and right side similar.

    Install the BSF as indicated.For additional information, refer to: Roof Panel - Vehicles Without: Roof Opening Panel (501-28 Roof Sheet Metal Repairs, Removal and Installation).

All vehicles

  1. Install the SPRs as indicated.
  2. Remove any excess adhesive.
  3. Make sure that any open or exposed panel joints are correctly sealed.
  4. Make sure that all installed SPR's are sealed using a Jaguar recommended sealer.
  5. Make sure corrosion protection is applied to all areas affected by repair.
  6. The installation of associated panels and components is the reversal of removal procedure.
