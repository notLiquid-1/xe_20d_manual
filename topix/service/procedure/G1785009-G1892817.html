  1. NOTE:

    The inner quarter panel upper is serviced as part of the inner quarter panel assembly.

    The inner quarter panel upper is serviced as indicated.
  2. Before commencing this procedure make sure that you are aware of all Health and Safety requirements.For additional information, refer to: Health and Safety Precautions (100-00 General Information, Description and Operation).
  3. For further information on the methods, tools and fixings used in this procedure refer to the body repairs - general information section.
    4. NOTES:

    • Where breakstem fasteners (BSF) removal is instructed remove fastener shaft with a 4mm punch and remove the head with a 6.5mm drill bit. Clean all the drilled holes to make sure they are free from debris.

    • Where self piercing rivets (SPR's) removal is instructed use either the approved SPR tool or 5.3mm drill bit.

    • Any joints sealed as part of vehicle manufacture must be sealed as part of the repair.

Installation
CAUTION:

Where structural adhesive is to be applied between surfaces it is essential compression is applied to the joints during the curing period.

NOTE:

BSF installation involves using the Jaguar approved BSF installation tool and requires a hole diameter of 6.5mm.

SPR installation requires using the Jaguar approved SPR tool.

  1. Remove the rivet remnants.
  2. Dress flanges where necessary.
  3. Using a fine bristle disc clean and prepare the panel surfaces.
  4. Remove the SPR's and detach the inner quarter panel upper from the service assembly as indicated.
  5. Offer up the new inner quarter panel upper and clamp into position. Check alignment if correct proceed to next step, if not rectify and recheck before proceeding.
  6. Drill holes where the BSF's are to be installed as indicated.
  7. Drill holes where the BSF's are to be installed as indicated.
  8. Drill holes where the BSF's are to be installed as indicated.
  9. Remove the new inner quarter panel upper.
  10. Deburr the drilled holes.
  11. Using a fine bristle disc clean and prepare the panel surfaces.
  12. Apply the coupling agent where the Jaguar recommended bonding material is to be applied and allow to dry.
  13. NOTE:

    Make sure a continuous bead of adhesive surrounds fixing holes.

    Apply a 5mm zig zag bead of the Jaguar recommended bonding material as indicated.
  14. Offer up the new inner quarter panel upper and clamp into position. Check alignment if correct proceed to next step, if not rectify and recheck before proceeding.
  15. Install the SPR's and the BSF's as indicated.
  16. Install the BSF's as indicated.
  17. Install the BSF's and the bolts and tighten the bolts to 25Nm as indicated.
  18. Remove any excess adhesive.
  19. Using the old quarter panel for reference, mark and cut the new quarter panel where the MIG welded butt joints are to be made as indicated.
  20. Measure and cut a 50mm wide backing plates, 25mm each side of the MIG butt joint, from the quarter panel remnant.
  21. Cut two run on/run off tabs from the quarter panel remnant.
  22. Offer up the new quarter panel upper and clamp into position. Check alignment if correct proceed to next step, if not rectify and recheck before proceeding.
  23. Drill holes where the backing plate is to be installed as indicated.
  24. Remove the quarter panel.
  25. Deburr the drilled holes.
  26. Using a fine bristle disc clean and prepare the panel surfaces.
  27. Install and MIG plug weld the backing plate as indicated.
  28. NOTES:

    • Make sure that these SPR's are installed before the installation of the quarter panel.

    • When the quarter panel is installed, make sure that the area around the SPR's is clamped until the bonding material has cured.

    Install the SPR's as indicated.
  29. Apply the coupling agent where the Jaguar recommended bonding material is to be applied to the quarter panel and allow to dry.
  30. NOTE:

    Make sure a continuous bead of adhesive surrounds fixing holes.

    Apply a 5mm zig zag bead of the Jaguar recommended bonding material to the new quarter panel.
  31. Offer up the new quarter panel and clamp into position. Check alignment if correct proceed to next step, if not rectify and recheck before proceeding.
  32. Tack MIG weld the butt joint as indicated.
  33. Install the SPR's as indicated.
  34. MIG plug weld the backing plate as indicated.
  35. Install the run on/off tabs and MIG weld the butt joint as indicated.
  36. Continue with the installation of the quarter panel following the quarter panel installation procedure.
  37. Dress all welded joints.
  38. Remove any excess adhesive.
  39. Cut off the run on/run off tabs.
  40. Carry out a non destructive crack inspection on the MIG butt joints. If correct proceed to next step, if not rectify and recheck before proceeding.
  41. Make sure that any open or exposed panel joints are correctly sealed.
  42. Make sure that all installed SPR's are sealed using a Jaguar recommended sealer.
  43. Make sure corrosion protection is applied to all areas affected by repair.
  44. The installation of associated panels and components is the reversal of removal procedure.
