Removal
CAUTION:

Use a suitable trim tool.

NOTES:

  • This procedure contains some variation in the illustration depending on the vehicle specification, but essential information is always correct.

  • This procedure contains illustrations showing certain components removed to provide extra clarity.


  1. Release the front door interior handle trim.
  2. Remove the bolt.
    • Remove the access cover.
    • Remove the bolt.
    • Remove the bolt.
    • Remove the door lock/unlock switch.
    • Disconnect the 2 electrical connectors.
  5. Remove the 2 center bolts.
    • Release the 10 plastic clips.
    • Release the 5 clips at the top of the door trim panel.
    • Release the door trim panel inner handle cable.
    • Remove the front door trim panel.
  7. NOTE:

    Only do the following step(s) if a new component is to be installed.

    • Remove the 5 screws.
    • Remove the door trim panel inner handle.
    • Release the 6 clips.
    • Remove the driver door switchpack.
Installation
  1. Install the driver door switchpack.
    • Install the door trim panel inner handle.
    • Install and tighten the 5 screws.
      Torque: 2.5 Nm
    • Install the door trim panel inner handle cable.
    • Install the front door trim panel.
    • Secure the 5 clips at the top of the door trim panel.
    • Secure the 10 plastic clips.
  4. Install and tighten the 2 center bolts.
    Torque: 2.5 Nm
    • Connect the 2 electrical connector.
    • Install the door lock/unlock switch.
    • Install and tighten the bolt.
      Torque: 2.5 Nm
    • Install and tighten the bolt.
      Torque: 2.5 Nm
    • Install the access cover.
  7. Install and tighten the bolt.
    Torque: 2.5 Nm
  8. Install the front door interior handle trim.
