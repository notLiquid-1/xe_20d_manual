OVERVIEW

The navigation system provides audible and visual route guidance information to enable the driver to reach a desired destination. The system allows the driver to choose the route using minor or major roads or highways with the option different routes according to market specification (e.g. recommended, shortest, fastest, and most economical). Directions to hospitals, museums, monuments and hotels are also available.

The navigation system is integrated with the audio system and shares a number of components common to all systems.

The message center in the Instrument Cluster (IC) and Head Up Display (HUD) (if equipped) can be set up to show navigation instructions. For additional information, refer to: Instrument Cluster (413-01 Instrument Cluster, Description and Operation).

Navigation System

The navigation system uses the following components:

  • Infotainment Master Controller (IMC)
  • Touchscreen (TS)
  • Roof pod GPS antenna
  • Instrument Cluster - virtual display (IC)
  • Head Up Display (HUD) (if fitted).
NOTE:

The InControl Touch Pro navigation system does not support voice control for navigation

Two navigation systems are available with the InControl Touch Pro navigation system:

  • Incontrol Touch Pro Solid State Drive (SSD) navigation
  • InControl Pro 'Connected Navigation' - only available when InControl Connect Pro Pack option is installed.

InControl Touch Pro - SSD Navigation System

Map data is stored on the SSD within the Infotainment Master Controller (IMC) providing faster access and update times for map data and display. Map updates are provided via the Universal Serial Bus (USB) connection in the front portable media interface panel.

InControl Pro Connected Navigation System

In addition to the InControl Pro SSD navigation system, Connected Navigation is available as a feature of InControl Pro Services. The connected navigation system is embedded within the Infotainment Master Controller (IMC) software and offers a mobile and web companion application for when drivers leave their vehicle.

Connected Navigation has the following features:

  • My Commute
  • Real time traffic flow
  • Fuel prices services
  • Approach mode
  • Online search
  • Safety cameras
  • Satellite views
  • Online routing
  • Profiles
  • Door to door routing
  • Sharing.

The Infotainment Master Controller (IMC) is connected to other vehicle systems via the High Speed (HS) Controller Area Network (CAN) comfort systems bus and via an Automotive Pixel Link 2 (APIX2) connection to the Touchscreen (TS). The driver can control navigation system functions by using the TS. The TS sends a control request signal to the IMC via the APIX2 connection, where the requested control information is processed.

The message center as part of the Instrument Cluster (IC) is connected to the IMC via HS CAN comfort systems bus.

If the vehicle is equipped with Head Up Display (HUD), the IMC is connected to the Head Up Display Control Module (HUDCM) via the HS CAN comfort systems bus. This connection allows turn by turn navigation data to be displayed on the windshield by the HUDCM. For additional information, refer to: Instrument Cluster (413-01 Instrument Cluster, Description and Operation).

Navigation data is stored on the Solid State Drive (SSD) within the IMC. Map updates can be downloaded and the system updated via a Universal Serial bus (USB) connection through the front portable media interface panel. The SSD is also used by other systems to store data and media.

NOTE:

Due to the map data file size, it is recommended that a USB 3.0 memory stick is used to update the map data. For further information on Navigation system updates, please visit the Map Updates section on www.jaguar.com/navigation, or contact a Jaguar Retailer.

The InControl Touch Pro navigation system incorporates a 'dead reckoning' system. For times when the Global Positioning Signal (GPS) is lost, the IMC uses various vehicle sensors, navigation map, odometer, 3D gyroscope and a 3D accelerometer (integrated in the IMC) and reverse gear signal to determine the vehicle location and the direction of travel.

Voice Control System

The voice control system provides the driver with the option of voice control for a range of supported functions.

Navigation user voice commands are made using the Jaguar Voice system. The Infotainment Master Controller (IMC) processes the Local Interconnect Network (LIN) signal from the Voice switch on the left steering wheel switchpack to initiate Jaguar Voice.

For additional information, refer to: Voice Control (415-01A Information and Entertainment System - Vehicles With: InControl Touch Pro, Description and Operation).

Roof Pod

The navigation GPS antenna is located in the roof pod. Depending on vehicle specification the roof pod can contain digital radio, Global System for Mobile communications (GSM) or satellite radio (SDARS) antennas.

The roof pod is fitted externally at the rear of the roof panel. Two co-axial connectors provide for the connection for the GPS antenna, the GSM antenna and the digital radio antenna.

On vehicles with the InControl Touch Pro Connected Navigation and/or the Telematics system, the GPS/GSM signal is passed via a co-axial cable to the GPS signal splitter where it is then passed to the Infotainment Master Controller (IMC) for use by the navigation system and to the Telematics Control Module (TCU) for use by the telematics system.

For additional information, refer to: Telematics (415-01A Information and Entertainment System - Vehicles With: InControl Touch Pro, Description and Operation).

Vehicle Information and Communication System Beacon Antenna – Japan Only

Vehicle Information and Communication System (VICS) infrared transmissions are received by the VICS beacon antenna, located on the top of the instrument panel on the left side. Power supply for the VICS beacon antenna is provided by the electronic toll collection module. The electronic toll collection module transmits the data received from the VICS beacon antenna to the Infotainment Master Controller (IMC) via Universal Serial Bus (USB) connection. The received data messages are used by the IMC for navigation.

Electronic Toll Collection Module Antenna – Japan Only

The electronic toll collection module antenna is located on the top of the instrument panel on the right side and it is connected directly to the electronic toll collection module. The electronic toll collection module antenna wirelessly communicates through microwave Radio Frequency (RF) transmission signals with a Dedicated Short Range Communication (DSRC) system. These signals are transmitted with the antenna located at the toll gates.

The DSRC system also provides VICS transmission with antennas located along the expressways. These signals are transmitted to the IMC via the electronic toll collection module through USB connection.

Electronic Toll Collection Module – Japan Only

The electronic toll collection system comprises an electronic toll collection module and a electronic toll collection module antenna. The electronic toll collection module is located in the glovebox. The electronic toll collection module receives a fused power supply via a relay in the Passenger Junction Box (PJB).

The electronic toll collection system credit card is inserted into the electronic toll collection module which reads the data on the card. The electronic toll collection module sends and receives vehicle information necessary for paying the appropriate toll fees.

The electronic toll collection module is connected to the USB 2.0 hub and can pass relevant information directly to the Infotainment Master Controller (IMC).

OPERATION

Navigation system

The navigation system receives Global Navigation Satellite System (GNSS) information via the Global Positioning System (GPS) antenna located in the roof pod. The GNSS signals are used by the navigation system to calculate the vehicle’s position. Once the driver has input a desired destination, the navigation system can calculate a route, based on the driver's pre-determined preferences or the default settings in the navigation system. The navigation system is accessed from the Touchscreen (TS) home menu.

The GNSS signal is available to the navigation system at all times when the vehicle ignition is switched on.

Navigation is initiated by the driver inputting a destination. This can be achieved via the TS by:

  • Entering an address
  • Entering a post code
  • Choosing a previous destination
  • Choosing a point of interest from the map database
  • Choosing the home location
  • Choosing a memory stored location.

On vehicles with InControl Touch navigation system, the driver can also enter the destination by the following:

  • Choosing town center
  • Entering geo-coordinates
  • Choosing trips
  • Choosing a point on map
  • Entering phone number.

The driver is then guided to the destination by scrolling map display and voice guidance. Navigation instruction can be displayed on the message center in the Instrument Cluster (IC) or the Head Up Display (HUD). The Infotainment Master Controller (IMC) sends signals on the High Speed (HS) Controller Area Network (CAN) comfort systems bus to the IC or to the Head Up Display Control Module (HUDCM).

Voice Guidance

The voice signals are passed from the Infotainment Master Controller (IMC) to the AAM via BroadRReach® Ethernet connection.

The navigation audio output is through the front speakers whilst the background audio, for example radio or external media, is played at a reduced volume on the rear speakers.

Vehicle Information and Communication System - Japan Only

The Vehicle Information and Communication System (VICS) is a unique system to Japan and provides countrywide coverage and broadcasts of real-time traffic and weather information.

The VICS has three methods of transmitting the traffic data to the vehicle navigation system:

  • Frequency Modulation (FM)
  • microwave Radio Frequency (RF)
  • Infrared.

The infrared transmissions are received by the VICS beacon antenna located on the top of the instrument panel on the left side.

The microwave RF transmission signals are received by the Electronic Toll Collection Module antenna which communicates on the 5.8 GHz wireless band with a Dedicated Short Range Communication (DSRC) system with the antennas located along the expressways. The information is passed to the Infotainment Master Controller (IMC) via the electronic toll collection module.

Additional information is also transmitted on a FM wavelength and is received by the FM antenna. The received FM signal is then passed to the IMC, via an FM antenna amplifier.

For additional information, refer to: Antenna (415-01A Information and Entertainment System - Vehicles With: InControl Touch Pro, Description and Operation).

The VICS supplies information to enable the IMC to re-route the navigation guidance or to inform the vehicle driver of traffic conditions in the vehicles vicinity.

The RF transmissions are generally transmitted from road side beacons mainly on expressways.

The information transmitted is as follows:

  • Traffic congestion
  • Travel time to next intersection
  • Traffic conditions in surrounding areas and expressway turn offs
  • Traffic accidents
  • Speed limits
  • Lane regulations
  • Tire change
  • Parking availability at expressway service areas and parking areas.

Infrared transmissions are transmitted from road side beacons on major trunk roads.

The information transmitted is as follows:

  • Traffic congestion and travel time
  • Traffic accidents
  • Breakdowns
  • Road works restrictions
  • Parking availability.

FM transmissions are broadcast as part of the FM multiplex broadcasting system from NHK (Japanese Broadcasting Corporation) FM stations.

The information transmitted is:

  • Traffic congestion and travel time for wide areas
  • Traffic accidents, road works, speed limits and lane restrictions for a wide area
  • Parking availability information.

The traffic data is split from the normal FM transmissions by the FM antenna amplifier.

Electronic Toll Collection System - Japan Only

An electronic toll collection system credit card is inserted into the electronic toll collection module which reads the data on the card. The electronic toll collection module is connected to the electronic toll collection module antenna which wirelessly communicates via a Dedicated Short Range Communication (DSRC) system with the antenna located at the toll gates.

Dedicated Short Range Communication (DSRC) is a system which makes bi-directional communication, within a short distance via radio beacons, on the 5.8 GHz wireless band. DSRC communications enable high-speed, high-capacity data transmission and reception. The system also features high-level security to ensure the safety of the card user's private information.
