Inspection
NOTES:

  • Some variation in the illustrations may occur, but the essential information is always correct.

  • Engine shown removed for clarity.


  1. WARNING:

    Make sure to support the vehicle with axle stands.

    Raise and support the vehicle.
  2. Refer to: Oil Pan (303-01B Engine - GTDi 2.0L Petrol, Removal and Installation).
  3. Torque: 25 Nm
  4. Install the special tool.
    Special Tool(s): JLR-303-748
  5. Rotate the engine clockwise until the crankshaft stops against the special tool as indicated.
  6. NOTE:

    Note the orientation of the alignment marks

    Mark the balance shaft assembly at the positions illustrated, to show the assembly at top-dead centre (TDC).
  7. NOTE:

    Remove and discard the O-ring seal.

    Remove the oil strainer.
  8. Remove the balance shaft assembly.
  9. NOTE:

    Install a new balance shaft assembly is there is any damage.

    Remove the original shims and visually inspect the balance shaft assembly for damage and smooth operation.
Check
  1. Install the master adjustment shims (1.50mm).
  2. Make sure that the alignment marks and marks on the balance shaft assembly are correctly aligned to show TDC.
  3. NOTE:

    Tighten the retaining bolts in the sequence illustrated.

    Install the balance shaft assembly
    Torque:
    Stage 1: 25 Nm
    Stage 2: 42 Nm
  4. Remove the special tool.
    Special Tool(s): JLR-303-748
  5. Rotate the engine clockwise to confirm that the balance shaft assembly is meshed correctly with the crankshaft gear.
  6. Install the special tool.
    Special Tool(s): JLR-303-748
  7. Rotate the engine clockwise until the crankshaft stops against the special tool as indicated.
  8. CAUTIONS:

    • Make sure that the DTI gauge is located and supported with stand that is secure to avoid any movement while obtaining the values.

    • Make sure that the clamp and the Allen key do not come into contact with the balance shaft housing.

    1. Using a suitable retaining clamp, install a 5mm Allen key in the position shown.
    1. Install the DTI gauge to a suitable secured base, at the height illustrated.
      General Equipment: Dial Test Indicator (DTI) gauge
  9. Remove the special tool.
    Special Tool(s): JLR-303-748
  10. CAUTION:

    Using a suitable screwdriver, apply and hold force to the crankshaft in the position illustrated, to allow correct values to be taken.

    • NOTES:

      • To give accurate values, the measurements must be taken at six positions (10°, 30°, 100°, 190°, 210° and 280°)

      • Make sure that the DTI gauge is set to zero after each measurement.

      Rotate the engine clockwise while taking the measurements.
    • Reference the values against the backlash specifications and make sure that the correct shims are installed to the balance shaft assembly on installation.
      Refer to: Specifications (303-01B Engine - GTDi 2.0L Petrol, Specifications).
    • If the backlash measurements exceed 0.101mm, then a new balance shaft assembly must be installed.
      Refer to: Balance Shaft Assembly (303-01B Engine - GTDi 2.0L Petrol, Removal and Installation).
Adjustment
  1. Remove the balance shaft assembly.
  2. NOTE:

    Install a new balance shaft assembly is there is any damage.

    Remove the original shims and visually inspect the balance shaft assembly for damage and smooth operation.
  3. Install the correct shims from the specifications table.
  4. Install the special tool.
    Special Tool(s): JLR-303-748
  5. Rotate the engine clockwise until the crankshaft stops against the special tool as indicated.
  6. Install the balance shaft assembly retaining bolts finger tight.
  7. Make sure that the alignment marks and marks on the balance shaft assembly are correctly aligned to show TDC.
  8. NOTE:

    Tighten the retaining bolts in the sequence illustrated.

    Install the balance shaft assembly
    Torque:
    Stage 1: 25 Nm
    Stage 2: 42 Nm
  9. Remove the special tool.
    Special Tool(s): JLR-303-748
  10. Torque: 25 Nm
  11. NOTE:

    Make sure a new O-ring seal is installed.

    Install the oil strainer.
    Renew Part: Oil pick up pipe O-ring seal Quantity: 1.
    Torque: 10 Nm
  12. Refer to: Oil Pan (303-01B Engine - GTDi 2.0L Petrol, Removal and Installation).
