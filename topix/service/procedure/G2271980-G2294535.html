Exterior Door Handle Assembly (G2271980)

GENERAL PROCEDURES

Activation
CAUTION:

Do not dismantle pre-assembled exterior door handles before painting as this will cause damage to the weatherproof seals.

NOTES:

  • This procedure contains some variation in the illustrations depending on the vehicle specification, but the essential information is always correct.

  • This procedure contains illustrations showing certain components removed to provide extra clarity.


  1. NOTE:

    This step only applies to chrome handles with painted inserts.

    • 1) Install the exterior door handle insert into the exterior door handle cover.
    • 2) The painted exterior door handle insert is shown in the installed position.
    • 3) Inspect from the rear to make sure that all of the clips are correctly installed.
  2. Remove the backing from the tape on the electronic unit as illustrated.
  3. Install the electrical connector through the exterior door handle as illustrated.
  4. NOTE:

    Make sure the electrical connector is installed to the correct orientation.

    Install the electrical connector into the exterior door handle as illustrated.
  5. CAUTION:

    Make sure the self adhesive tape is bonded correctly.

    • Install the antenna and electronic unit into the exterior door handle, align the clip as illustrated.
    • Firmly press the electronic unit into the exterior door handle.
  6. Remove the backing from the tape on the electronic unit.
    • Assemble the exterior door handle by clipping the 2 sections together as illustrated.
    • 1) Hook the outer exterior door handle cover around the end of the inner exterior door handle strap.
    • 2) Rotate the outer exterior door handle cover to install onto the inner exterior door handle cover.
    • Firmly squeeze the exterior door handle assembly at the electronic unit end to make sure the self adhesive tape has bonded correctly.
  8. Install and tighten the 2 exterior door handle bolts.
  9. Install the exterior door handle gasket as illustrated.
  10. Make sure the wiring harness is positioned as illustrated.
    • Install the exterior door handle gasket.
    • Make sure the exterior door handle gasket is correctly installed onto the 6 locating pins.
  12. Make sure the exterior door handle gasket is installed correctly.
