Removal
NOTE:

Some variation in the illustrations may occur, but the essential information is always correct.


  1. Disconnect the startup battery ground cable.
    Refer to: Startup Battery Disconnect and Connect (414-01 Battery, Mounting and Cables, General Procedures).
  2. WARNING:

    Make sure to support the vehicle with axle stands.

    Raise and support the vehicle.
    Refer to: Lifting (100-02 Jacking and Lifting, Description and Operation).
    • Remove the selector cable.
    • Secure with a cable tie.
  4. CAUTION:

    Care must be taken not to damage the component.

    Secure with cable ties.
  5. CAUTION:
    • Make sure that the driveshaft is supported with suitable retaining straps.
    • Discard the bolts.
    Remove the driveshaft to transmission output flange bolts.
    • Remove the cable tie.
  7. Remove the transmission support insulator.
    Refer to: Transmission Support Insulator (307-01 Automatic Transmission/Transaxle - Vehicles With: 8HP45 8-Speed Automatic Transmission RWD, Removal and Installation).
  8. Remove the damper.
  9. CAUTION:

    Discard the nut.

    • Remove the output flange retaining nut.
      Special Tool(s): 205-053
  10. CAUTION:

    Discard the output flange.

    Using the special tool, remove the output flange.
    Special Tool(s): 303-D121
  11. CAUTION:

    Discard the seal.

    Using the special tool, remove the extension housing seal.
    Special Tool(s): 100-012, 100-012-01, 308-375
Installation
  1. CAUTION:
    • Make sure that the mating faces are clean and free of foreign material.
    • Install a new seal.
    • Using the special tool, install the extension housing seal.
      Renew Part: Extension housing seal Quantity: 1.
    • Special Tool(s): 307-520
    • Special Tool(s): JLR-307-520-01
  2. Install the output flange.
    Renew Part: Transmission output shaft flange Quantity: 1.
  3. WARNING:

    Make sure that a new nut is installed.

    • Using the special tool, install the transmission output flange retaining nut.
      Renew Part: Output shaft flange nut Quantity: 1.
      Torque: 60 Nm
    • Special Tool(s): 205-053
  4. Torque: 48 Nm
  5. CAUTION:

    Care must be taken not to damage the component.

    • Secure with a cable tie.
  6. CAUTION:

    Make sure that new bolts are installed.

    Install the driveshaft.
    Renew Part: Driveshaft to front output flange bolts Quantity: 6.
    Torque: 75 Nm
    • Remove the cable tie.
  8. Install the selector cable.
    Torque: 11 Nm
  9. Install the transmission support insulator.
    Refer to: Transmission Support Insulator (307-01 Automatic Transmission/Transaxle - Vehicles With: 8HP45 8-Speed Automatic Transmission RWD, Removal and Installation).
  10. Connect the startup battery ground cable.
    Refer to: Startup Battery Disconnect and Connect (414-01 Battery, Mounting and Cables, General Procedures).
