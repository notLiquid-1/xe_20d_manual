Removal
CAUTIONS:

  • Lay the glass on felt covered supports. Do not stand on edge as this can cause chips which subsequently develop into cracks.

  • Make sure to protect the paintwork.

NOTE:

Removal steps in this procedure may contain installation details.

  2. NOTE:

    Repeat the step for the other side.

    Refer to: A-Pillar Trim Panel (501-05 Interior Trim and Ornamentation, Removal and Installation).
  3. Refer to: Rain Sensor (501-16 Wipers and Washers, Removal and Installation).
  10. NOTES:

    • Repeat the step for the other side.

    • If the A pillar trim clips are damaged, install new clips.

    Remove the A pillar trim panel.
  11. CAUTIONS:

    • Protect the surrounding components.

    • Protect the surrounding paintwork to avoid damage.

  12. WARNING:

    This step requires the aid of another technician.

Installation
  1. CAUTION:

    Make sure that the mating faces are clean and free of foreign material.

  2. CAUTION:

    Touching the adhesive surface will impair re-bonding.

    NOTE:

    Make sure that a Hi-modulus sealer (for example, Wurth Ultimate adhesive) is used when bonding the component to the vehicle.

  3. CAUTION:

    Make sure that equal pressure is applied to the full length of the component.

    NOTE:

    This step requires the aid of another technician.

  4. NOTE:

    Make sure that the gap around the component is even.

  5. CAUTION:

    Make sure that no excess sealant residue is evident.

    • If water is used as a means for the leak check, then allow sealant to dry before testing.
    • Spray water around the windshield glass, mark any area that leaks. Dry the windshield glass and sealant before applying additional sealant.
  6. NOTES:

    • Repeat the step for the other side.

    • If the A pillar trim clips are damaged, install new clips.

    Install the A pillar trim panel.
  13. Refer to: Rain Sensor (501-16 Wipers and Washers, Removal and Installation).
  14. NOTE:

    Repeat the step for the other side.

    Refer to: A-Pillar Trim Panel (501-05 Interior Trim and Ornamentation, Removal and Installation).
  15. Refer to: Cowl Vent Screen (501-02 Front End Body Panels, Removal and Installation).
  16. Following a windshield replacement and installation of the original image processing module (IPMA), the vehicle must be driven above 65kph / 42mph to activate the system for self calibration. The vehicle then must be driven for a minimum of 5 minutes at speeds above 30kph / 20mph on normal roads to self calibrate the system. If the self calibration is unsuccessful the DTC C1001-78 will be logged, at which point the vehicle will need to be re-calibrated using Jaguar Land Rover approved diagnostic equipment.
