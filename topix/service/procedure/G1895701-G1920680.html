Removal
NOTE:
  • This procedure contains some variation in the illustrations depending on the vehicle specification, but the essential information is always correct.
  • This procedure contains illustrations showing certain components removed to provide extra clarity.

  1. Raise and support the vehicle on a suitable 2 post lift.
    Refer to: Lifting (100-02 Jacking and Lifting, Description and Operation).
  2. Remove the front halfshaft.
    Refer to: Front Halfshaft (205-04 Front Drive Halfshafts, Removal and Installation).
  3. Remove and discard the 2 retaining clips.
  4. CAUTION:

    Care must be taken not to damage the component.

    Make sure the halfshaft is positioned as illustrated.
  5. CAUTION:

    Using a suitable soft drift, strike the joint in the area indicated to release it from the shaft.

    Using a soft hammer remove the inner joint
  6. CAUTION:

    Using a suitable soft drift, strike the component in the direction shown to allow easy release of the retaining clip.

    Using a suitable tool mark the position of the component.
  7. Remove and discard the retaining clips.
  8. Remove the bearings using a suitable puller.
    Special Tool(s): 303-D121
  9. Remove and discard the inner Constant Velocity (CV) joint boot.
  10. Remove and discard the 2 retaining clips and the outer CV boot.
Installation
    • Clean and inspect the components for deterioration.
    • Pack the outer Constant Velocity (CV) joint with the grease supplied.
    • Install the outer CV boot.
  2. Make sure the outer CV boot is correctly fitted.
  3. NOTE:

    Make sure the retaining clip is correctly seated.

    Install the 2 retaining clips using a suitable tool.
  4. NOTE:

    Use new retaining clips.

    Install the inner CV boot and retaining clip.
    Renew Part: Inner constant velocity joint boot kit Quantity: 1.
  5. Install the retaining clip.
    Renew Part: Circlip Quantity: 1.
  6. Install the inner CV joint.
  7. Install the 2 retaining clips using a suitable tool.
  8. Install the front halfshaft.
    Refer to: Front Halfshaft (205-04 Front Drive Halfshafts, Removal and Installation).
